# CDN-source

使用免费的CDN平台 **`jsDelivr`** 搭建个人CDN资源网盘.

访问资料方式:
```
https://cdn.jsdelivr.net/gh/你的用户名/你的仓库名@发布的版本号/文件夹名/文件名

// 例如
https://cdn.jsdelivr.net/gh/Douc1998/CDN-source/image/myLogo.png
```

(版本号可加可不加, 根据情况而定)
